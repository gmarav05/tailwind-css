## Landing page
A simple clean Landing Page built using modern frontend technology.

The goal of this project is to practice layout design, responsiveness, and utility-first styling using **Tailwind CSS**.
  
Built with simple HTML5 and **Tailwind CSS**.

## Features

- Modern UI with Tailwind CSS.

- Clean and minimal code structure.

- **Simple UI** - Input text, select target languages, click translate.

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile.


## Learnings

- Learned how to setup tailwind with CLI.

- Practiced Tailwind CSS.

- Practiced semantic and structured code.

- Practiced Grid layout using Tailwind CSS.

- Practiced more CSS flexbox.
